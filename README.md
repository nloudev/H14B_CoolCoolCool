# H14B_CoolCoolCool

Minimal Node.js + Express API with Swagger documentation.

## Run locally

1. Install dependencies:

	```bash
	npm install
	```

2. Start the server:

	```bash
	npm start
	```

3. Open:

- API health: `http://localhost:3000/health`
- Swagger UI: `http://localhost:3000/api-docs`
- OpenAPI JSON: `http://localhost:3000/openapi.json`
